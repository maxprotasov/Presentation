Плюсы:

  1. Легко построить любую сетку.
  2. Манипуляция Grid-элементом.

Термины:

  • grid container - грид-контейнер
  • grid item - грид-элемент
  • grid track - грид-полоса
  • grid cell - рид-ячейка
  • grid line - грид-линия
  • grid area - грид-область
